# Introduction

## Installation

Using npm:

```bash
npm install @loggists/event-tracker
```

Using yarn:

```bash
yarn add @loggists/event-tracker
```

Using pnpm:

```bash
pnpm add @loggists/event-tracker
```

## Quick Start

Here's a simple example of how to use event-tracker:

```tsx
import { createTracker } from "@loggists/event-tracker";

// Create a tracker instance
const [Track, useTracker] = createTracker({
  DOMEvents: {
    onClick: (params, context) => {
      console.log("Click event:", params, context);
    },
  },
});

// Use in your app
function App() {
  return (
    <Track.Provider initialContext={{ userId: "user-123" }}>
      <Track.Click params={{ buttonId: "submit" }}>
        <button>Submit</button>
      </Track.Click>
    </Track.Provider>
  );
}
```

## Basic Concepts

event-tracker is built around a few core concepts:

1. **Tracker Creation**: Use `createTracker` to create a tracker instance with your desired configuration
2. **Provider**: Wrap your app with `Track.Provider` to provide context
3. **Event Components**: Use components like `Track.Click` or `Track.Impression` to track events
4. **Hooks**: Use the `useTracker` hook to access tracking functionality in your components
