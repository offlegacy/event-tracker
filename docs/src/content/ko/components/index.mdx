# Provider

`Provider` 컴포넌트는 애플리케이션에 초기 컨텍스트를 연결하고 제공합니다.

```tsx
import { createTracker } from '@offlegacy/event-tracker'

const [Track] = createTracker({...})

function App() {
  return (
    <Track.Provider initialContext={{}}>
      {/* 애플리케이션 컨텐츠 */}
    </Track.Provider>
  )
}
```

### Props

- `initialContext?: Context` - 초기 컨텍스트 값
